Le repertoire par défaut sera celui de votre dépôt.
Vos réponses seront écrite en mode graphique (sans passer par le terminal) dans un fichier `tp.md`; commencer par créer ce fichier (graphiquement).

Par la suite, les créations de fichiers/dossiers se feront depuis le terminal.

### 1.
Créer un dossier `tpLinux`

### 2.
Utiliser une commande pour vérifier la création

### 3. 
Se déplacer dans le dosser `tplinux` et créer les fichiers:
    - fic1.txt
    - fic2.txt
    - fichier.md
    - image.jpg
    - exos.md
    - exoos.txt

### 4.
Créer l'arborescence de dossiers suivante:
```
.
|____dosA
|       |____dosB
|       |____dosC
|
|___dosD
```

### 5.
Déplacer tous les fichiers dont le nom commence par `fi` dans le dossier `dosB`

### 6.
Se déplacer dans `dosB`.

Copier respectivement les fichiers `fic1.txt`, `fic2.txt`, `fichier.md` dans `dosA`, `dosC`, `dosD`

### 7.
Revenir dans `tplinux`. 

Utiliser la commande `echo` pour ajouter le texte `"bonjour tout le monde"` dans le fichier `fic1.txt`

Afficher la taille de ce fichier.

### 8.
Rechercher dans l'arborescence, tous les fichiers contenant une extension d'exactement trois caractères.

### 9.
Renommer le fichier `exoos.txt` en `exos.txt`.

### 10.
Revenir dans le répertoire par défaut.

Créer un fichier `supprimer.sh` et ajouter dans ce fichier :
    - une commande qui supprime toute l'arboresence `tpLinux` 
    - la commande `tree`

Donner les droits d'éxécution à `user` puis exécuter le fichier `supprimer.sh`
